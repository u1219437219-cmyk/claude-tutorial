# Backend Review Skill

Review backend changes with a correctness and production-risk mindset.

## Focus

1. API contract risk
2. Data consistency
3. Error handling and logging
4. Concurrency / transaction / cache / message risks
5. Test gaps
6. Maintainability

## Output

- Findings first
- Why it matters in production
- Suggested fix or follow-up
- Validation notes
