# Backend Bugfix Skill

Fix backend bugs by identifying the root cause first.

## Focus

1. Failing behavior
2. Likely root cause
3. Relevant interfaces, services, jobs, logs, or data paths
4. Smallest safe fix
5. Validation and consistency risk
